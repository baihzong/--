复合数据类型

  • 列表、元组、字典和集合的关键字分别是：list、tuple、dict和set。这些关键字用于定义和创建不同类型的数据结构。

  • 列表

    • 列表（List）是可变的有序序列，用方括号 [] 来表示。列表中的元素可以修改，可以包含不同类型的元素，并且允许有重复的元素。

    • 切片

      • 通过执行切片操作来截取列表变量中的部分元素

      • c = [1, 2, 3, 4]
print(c[0:2])   #[1, 2]

      • 以下是各个数据类型是否支持切片操作的概述：

        1. 元组（tuple）：支持切片操作。可以通过切片操作获取元组中指定范围内的元素。

        2. 列表（list）：支持切片操作。可以通过切片操作获取列表中指定范围内的元素。

        3. 字符串（string）：支持切片操作。可以通过切片操作获取字符串中指定范围内的字符。

        4. 字典（dictionary）：不支持切片操作。字典是无序的键值对集合。

        5. 集合（set）：不支持切片操作。集合是无序、唯一的元素集合。

        6. my_tuple = (1, 2, 3, 4, 5)
my_list = [1, 2, 3, 4, 5]
my_string = "Hello, World!"

# 元组的切片操作
tuple_slice = my_tuple[1:4]
print(tuple_slice)   # (2, 3, 4)

# 列表的切片操作
list_slice = my_list[1:4]
print(list_slice)    # [2, 3, 4]

# 字符串的切片操作
string_slice = my_string[7:12]
print(string_slice)  # World

    • 列表函数和方法

      • 列表函数

        • len(list) 返回个数
        • max(list) 最大
        • min(list) 最小
        • list(seq) 将元组转换为列表

      • 列表方法

        • list.append(obj) 在列表末尾添加新的对象

        • list.pop(obj=list[-1]) 将对象插入列表

        • list.insert(index, obj) 移除列表中的一个元素(默认是最后一个元素)，并且返回该元素的值

        • list.remove(obj) 移除列表中的某个值的第一个匹配项

        • list.sort([func]) 对原列表进行排序

        • my_list = [3, 1, 4, 1, 5, 9]

# 在列表末尾添加新的对象
my_list.append(2)
print(my_list)   # [3, 1, 4, 1, 5, 9, 2]

# 将对象插入列表（在索引为2的位置插入元素6）
my_list.insert(2, 6)
print(my_list)   # [3, 1, 6, 4, 1, 5, 9, 2]

# 移除列表中索引为3的元素，并返回该元素的值
removed_element = my_list.pop(3)
print(my_list)           # [3, 1, 6, 1, 5, 9, 2]
print(removed_element)   # 4

# 移除列表中的某个值的第一个匹配项（移除值为1的第一个元素）
my_list.remove(1)
print(my_list)   # [3, 6, 1, 5, 9, 2]

# 对列表进行排序（默认升序）
my_list.sort()
my_list.sort(reverse=True) #降序
print(my_list)   # [1, 2, 3, 5, 6, 9]

  • 元组

    • 元组（Tuple）是不可变的有序序列，用圆括号 () 来表示。元组的元素不能修改，可以包含不同类型的元素，并且允许有重复的元素。通常用于存储相关的值，例如 (x, y) 表示二维坐标。
    • 元组和列表区别
      • 可变性：
        • 列表（List）是可变的，意味着可以修改列表中的元素、添加新元素或删除现有元素。
        • 元组（Tuple）是不可变的，一旦创建后，就不能修改元组中的元素。
      • 语法表示：
        • 列表使用方括号 [] 来表示，元素之间用逗号 , 分隔。
        • 元组使用圆括号 () 来表示，元素之间也用逗号 , 分隔。

  • 字典

    • 字典（Dictionary）是无序的键值对集合，用花括号 {} 来表示。字典中的元素包含键和对应的值，通过键来访问值。字典中的键必须是唯一的，但值可以重复。

    • 字典遍历

      • 在Python中，可以使用循环遍历字典的键、值或者键值对。

        1. 遍历字典的键：
          使用字典的keys()方法可以获取字典中的所有键，然后可以使用for循环遍历这些键。
        my_dict = {"a": 1, "b": 2, "c": 3}

# 遍历字典的键
for key in my_dict.keys():
    print(key)  # a b c
        1. 遍历字典的值：
          使用字典的values()方法可以获取字典中的所有值，然后可以使用for循环遍历这些值。
        my_dict = {"a": 1, "b": 2, "c": 3}

# 遍历字典的值
for value in my_dict.values():
    print(value)   # 1 2 3
        1. 遍历字典的键值对：
          使用字典的items()方法可以获取字典中的所有键值对，然后可以使用for循环遍历这些键值对。
        my_dict = {"a": 1, "b": 2, "c": 3}

# 遍历字典的键值对
for key, value in my_dict.items():
    print(key, value)
    
# a 1
  b 2
  c 3

  • 集合 92页

    • 集合（Set）是无序的唯一元素的集合，用花括号 {} 来表示。集合中的元素不重复，可以用来执行集合间的运算，例如并集、交集和差集等。

    • 集合的数学运算

      • 在Python中，可以使用集合进行一些数学运算，如并集、交集、差集和对称差等。以下是一些代码示例：

        set1 = {1, 2, 3, 4, 5}
set2 = {4, 5, 6, 7, 8}

# 并集
union_set = set1.union(set2)
print(union_set)  # {1, 2, 3, 4, 5, 6, 7, 8}

# 交集
intersection_set = set1.intersection(set2)
print(intersection_set)  # {4, 5}

# 差集
difference_set = set1.difference(set2)
print(difference_set)  # {1, 2, 3}

# 对称差
symmetric_difference_set = set1.symmetric_difference(set2)
print(symmetric_difference_set)  # {1, 2, 3, 6, 7, 8}

        在上述代码中，set1和set2是两个集合。可以使用集合对象的相关方法来执行不同的数学运算。

        • union()方法返回两个集合的并集。
        • intersection()方法返回两个集合的交集。
        • difference()方法返回第一个集合相对于第二个集合的差集，即只包含在第一个集合中而不包含在第二个集合中的元素。
        • symmetric_difference()方法返回两个集合的对称差集，即只包含在其中一个集合中而不包含在两个集合中的元素。

while guess != secret and count < 3:
    guess = int(input("错了，请再输入一次："))
    count += 1
    if guess == secret:
        print("对了")
    else:
        if guess > secret:
            print("大了")
        else:
            print("小了")

if count == 3 and guess != secret:
    print("猜测次数用完，游戏结束")
